Политика конфиденциальности

Дата вступления в силу: 1 января 2024

Последнее обновление: 1 января 2024

1. Общая информация

Приложение "Cafe Management App" ("мы", "нас", "наше") уважает вашу конфиденциальность и обязуется защищать ваши личные данные. Эта политика конфиденциальности объясняет, как мы собираем, используем и защищаем вашу информацию при использовании нашего приложения для управления рецептами и ингредиентами.

2. Собираемые данные

2.1 Данные учетной записи

  • Email адрес - для регистрации, входа в систему и восстановления доступа
  • Имя пользователя - для идентификации в системе
  • Роль пользователя - менеджер или сотрудник для определения прав доступа

2.2 Данные приложения

  • Рецепты - названия, инструкции, фотографии рецептов
  • Ингредиенты - названия, фотографии, единицы измерения
  • Теги и категории - для организации рецептов
  • Объемы порций - размеры порций для каждого рецепта
  • Количества ингредиентов - для каждого объема порции
  • Настройки пользователя - предпочтения и конфигурации

2.3 Технические данные

  • IP адрес - для обеспечения безопасности и предотвращения злоупотреблений
  • Информация об устройстве - тип устройства, операционная система, версия приложения
  • Логи использования - для диагностики проблем и улучшения приложения

3. Как мы используем данные

  • Предоставление услуг - для работы основных функций приложения
  • Аутентификация - для безопасного входа в систему
  • Управление доступом - для контроля прав доступа менеджеров и сотрудников
  • Улучшение приложения - для анализа использования и оптимизации
  • Техническая поддержка - для решения проблем пользователей
  • Безопасность - для защиты от мошенничества и злоупотреблений

4. Хранение и безопасность данных

Мы используем Supabase - надежную облачную платформу для хранения данных. Все данные:

  • Хранятся в защищенных центрах обработки данных
  • Шифруются в состоянии покоя и при передаче
  • Защищены от несанкционированного доступа
  • Регулярно резервируются

5. Передача данных

Мы не продаем, не обмениваем и не передаем ваши личные данные третьим лицам, за исключением:

  • Случаев, предусмотренных законом
  • Вашего явного согласия
  • Необходимости для предоставления услуг (например, хостинг через Supabase)

6. Ваши права

Вы имеете следующие права в отношении ваших данных:

  • Право на доступ - получить копию ваших данных
  • Право на исправление - исправить неточные данные
  • Право на удаление - удалить ваши данные
  • Право на ограничение - ограничить обработку данных
  • Право на переносимость - получить данные в структурированном формате

7. Удержание данных

Мы храним ваши данные только столько времени, сколько необходимо для:

  • Предоставления услуг приложения
  • Соблюдения юридических обязательств
  • Разрешения споров
  • Обеспечения безопасности

8. Изменения политики

Мы можем обновлять эту политику конфиденциальности время от времени. О значительных изменениях мы будем уведомлять вас через приложение или по email.
